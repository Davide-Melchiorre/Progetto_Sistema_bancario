# Sistema di Gestione Bancaria

Questo progetto consiste in una simulazione software di un sistema bancario completo, sviluppato interamente in Python. L'applicazione gestisce le operazioni quotidiane di una banca, coordinando le interazioni tra clienti, impiegati (cassieri e consulenti) e sportelli ATM, appoggiandosi a un database relazionale per la persistenza dei dati.

## Funzionalità Principali

Il sistema è suddiviso in moduli specifici in base al ruolo dell'utente:

### Area Cliente

* **Gestione Conti:** Visualizzazione del saldo (reale e disponibile) e lista dei movimenti.
* **Carte di Pagamento:** Gestione integrata di carte di debito e credito, con monitoraggio del plafond residuo.
* **Operazioni:** Esecuzione di bonifici verso altri conti e pagamenti rapidi.
* **Notifiche:** Sistema di avvisi per transazioni ricevute o effettuate.
